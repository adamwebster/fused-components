# Toggle

Toggle is useful for if you have two options that you would like the user to toggle between.  Simple put it is an on/off switch.
 
To change the state of the toggle update the active property to set to true or false.

## Default
<Canvas>
    <Story name="Toggle Unchecked">
    <>
    <Toggle aria-label="toggle"/>
    </>
    </Story>
</Canvas>

<Canvas>
    <Story name="Toggle Checked">
    <>
    <Toggle active aria-label="toggle"/>
    </>
    </Story>
</Canvas> 

## With Labels
<Canvas>
    <Story name="Toggle Unchecked Labeled">
    <>
    <Toggle showLabels aria-label="toggle"/>
    </>
    </Story>
</Canvas>

<Canvas>
    <Story name="Toggle Checked Labeled">
