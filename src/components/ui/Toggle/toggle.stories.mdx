# Toggle

## An toggle that has an active and not active state
 
To change the state of the toggle update the active property to set to true or false.

## Default
<Preview>
    <Story name="Toggle Unchecked" parameters={{ decorators: [withKnobs]}}>
    <>
    <Toggle aria-label="toggle"/>
    </>
    </Story>
</Preview>

<Preview>
    <Story name="Toggle Checked" parameters={{ decorators: [withKnobs]}}>
    <>
    <Toggle active aria-label="toggle"/>
    </>
    </Story>
</Preview>
