# Radio

The radio component is an replacement for the browsers radio element.  If the radio is checked or not is completely controlled by the consuming application through the **checked** property.

You can also use the <LinkTo kind="components-radio-group--radio-group">radio group</LinkTo> component to group multiple radios so only one is checked at once and add built in accessibility features.
