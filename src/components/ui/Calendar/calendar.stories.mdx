# Calendar

The calendar component can be used when you would like to show an calendar. It uses [dayjs](https://github.com/iamkun/dayjs) behind the scene to help with the date formatting.

If you use **onDateChange** it will pass back the formatted date in the specified format that you pass in the **dateFormat** property (defaults to MM/DD/YYYY).
