# Button

The button component is just that a button.  You can set the style of the button and put it into a loading state that will then disable the button and show the supplied loading icon. 

There are a few different button types to use; default, primary, and link button(set the **as** property to be a).

You can also supply your own button color to the button component.

You can also attach one of the many  <LinkTo kind="info-icons--page">icons</LinkTo> that are provided as part of the Fused Components library.
