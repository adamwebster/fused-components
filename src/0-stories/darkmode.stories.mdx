# Dark Mode

As of version 1.1.3 Fused Components now supports a dark mode.

## Using dark Mode

To use dark mode all you have to to do is wrap your code in the FCThemeProvider and pass the value of `theme: 'dark'`. Any of the components from this library that support dark mode will automatically be displayed with the dark mode applied.

### Example

    import React from 'react';
    import { FCThemProvider } from '@adamwebster/fused-components';

    const YourApp = () => {
      return(
        <FCThemeProvider value={{theme: 'dark'}} />
          // Other components go here
        <FCThemeProvider>
      )
    }

    export default YourApp;

You can access if the app is in dark mode by using the **useContext** hook to style any of your own components.

### Example of styling components using useContext

    import React, { useContext } from 'react';
    import { FCTheme } from '@adamwebster/fused-components';
    import styled from 'styled-components';

    const YourComponent = styled.div`
      background-color: ${props => props.theme === 'dark' ? '#000' : '#fff'}
    `
    const YourApp = () => {
      const theme = useContext(FCTheme);
      return(
        <YourComponent theme={theme?.theme}>
          // Other code can go here
        </YourComponent>
      )
    }

    export default YourApp;
